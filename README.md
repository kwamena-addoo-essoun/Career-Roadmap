# Career-Roadmap
Path to becoming a data engineer with gen AI

## 🗓️ Daily Time Budget Planner

A personal daily scheduler that combines career roadmap study tasks with recurring personal activities — all in one clean, interactive web app.

👉 **[Open the Daily Planner →](https://kwamena-addoo-essoun.github.io/Career-Roadmap/)**

Features:
- Color-coded daily time blocks (Sleep, Bible Studies, Piano, Career Study, Nephew Pickup, Free Time, Work)
- Interactive checkboxes that persist across page refreshes (localStorage)
- Dynamically displays the current week's study goal from the 16-week roadmap
- Week progress bar tracking your journey
- "Reset Day" button and motivational completion message
- Dark mode, mobile-friendly design
