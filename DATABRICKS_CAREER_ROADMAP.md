# 🗺️ Databricks Data Engineering Career Roadmap

## 🟢 PHASE 1: FOUNDATIONS
### *"Learn the language of data"* | Months 1–2

---

### Week 1 — SQL Basics
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Watch: *"SQL for Beginners"* on YouTube (Mosh Hamedani or freeCodeCamp)
- [ ] Complete: [Khan Academy SQL Course](https://www.khanacademy.org/computing/computer-programming/sql)
- [ ] Practice: `SELECT`, `WHERE`, `ORDER BY`, `GROUP BY`
- [ ] Mini challenge: Query a sample dataset and answer 5 questions from it
- [ ] **End of week goal:** Write a basic SQL query without help

---

### Week 2 — SQL Intermediate + Data Concepts
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Learn: `JOIN`, `SUBQUERIES`, `AGGREGATE FUNCTIONS`
- [ ] Practice on: [Mode SQL Tutorial](https://mode.com/sql-tutorial/)
- [ ] Learn: What is ETL? What is a data pipeline? What is a data warehouse?
- [ ] Watch: *"Data Engineering explained in 10 minutes"* on YouTube
- [ ] **End of week goal:** Understand what data engineers actually do day-to-day

---

### Week 3 — Python Basics (Part 1)
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Start: [freeCodeCamp Python for Beginners](https://www.freecodecamp.org/learn/scientific-computing-with-python/)
- [ ] Learn: Variables, data types, loops, `if/else` statements
- [ ] Practice: Write simple scripts (calculate totals, filter lists, print outputs)
- [ ] Tool: Use [Google Colab](https://colab.research.google.com/) — free Python environment, no install needed
- [ ] **End of week goal:** Write a Python script that reads a list and filters results

---

### Week 4 — Python Basics (Part 2)
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Learn: Functions, reading/writing files, working with lists and dictionaries
- [ ] Learn: Basic **Pandas** library (for working with data tables in Python)
- [ ] Practice: Load a CSV file, clean the data, print a summary
- [ ] Watch: *"Pandas for Beginners"* on YouTube (Corey Schafer)
- [ ] **End of week goal:** Load and manipulate a real dataset using Python + Pandas

---

## 🔵 PHASE 2: DATABRICKS & CERTIFICATION
### *"Get certified, get credible"* | Months 2–4

---

### Week 5 — Get Into Databricks
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Sign up: [Databricks Community Edition](https://community.cloud.databricks.com/) (100% free)
- [ ] Explore: The workspace, notebooks, and clusters
- [ ] Complete: [Databricks Lakehouse Fundamentals course](https://www.databricks.com/training/catalog) (free, ~3–4 hours)
- [ ] Run your first notebook: Load a CSV and query it with SQL inside Databricks
- [ ] **End of week goal:** Feel comfortable navigating Databricks

---

### Week 6 — Lakehouse Fundamentals Certification
> 🕐 Daily goal: 1–2 hours | 📍 Free

- [ ] Finish: All Lakehouse Fundamentals learning modules
- [ ] Revise: Key concepts — Delta Lake, Unity Catalog, Lakehouse architecture
- [ ] Take: The **Databricks Lakehouse Fundamentals exam** (FREE ✅)
- [ ] 🏆 Add the badge to your **LinkedIn profile** immediately
- [ ] **End of week goal:** Earn your 2nd Databricks credential 🎉

---

### Week 7 — Intro to Apache Spark & PySpark
> 🕐 Daily goal: 1.5–2 hours | 📍 Free

- [ ] Learn: What is Apache Spark? Why does Databricks use it?
- [ ] Watch: *"PySpark Tutorial for Beginners"* on YouTube (freeCodeCamp)
- [ ] Practice: Run basic PySpark commands in your Databricks Community notebook
- [ ] Learn: DataFrames, `read`, `write`, `filter`, `groupBy` in PySpark
- [ ] **End of week goal:** Read a dataset and run transformations using PySpark

---

### Week 8 — Delta Lake & Data Pipelines
> 🕐 Daily goal: 1.5–2 hours | 📍 Free

- [ ] Learn: What is Delta Lake? (Databricks' smart data storage format)
- [ ] Practice: Create a Delta table, update it, and query its history
- [ ] Learn: What is a data pipeline? Bronze → Silver → Gold architecture
- [ ] Use: [Databricks Academy free courses](https://academy.databricks.com/) on Delta Lake
- [ ] **End of week goal:** Build a simple Bronze → Silver data pipeline in Databricks

---

### Week 9 — Data Engineer Associate Exam Prep (Part 1)
> 🕐 Daily goal: 2 hours | 📍 ~$200 for exam when ready

- [ ] Enroll in: [Databricks Data Engineer Associate prep course on Udemy](https://www.udemy.com/courses/search/?q=databricks+data+engineer+associate) (~$15 on sale)
- [ ] Study: ETL with Spark, Delta Lake operations, workflows, and Auto Loader
- [ ] Practice: Databricks official sample exam questions
- [ ] Flashcards: Create revision cards for key concepts
- [ ] **End of week goal:** Complete 50% of exam prep material

---

### Week 10 — Data Engineer Associate Exam Prep (Part 2)
> 🕐 Daily goal: 2 hours

- [ ] Finish: All exam prep material
- [ ] Take: 2–3 full practice tests
- [ ] Revise: Any weak areas (check Databricks documentation)
- [ ] Review: Databricks Workflows, Jobs, and Cluster management
- [ ] **End of week goal:** Score 80%+ consistently on practice tests

---

### Week 11 — Take the Exam + Build Project #1
> 🕐 Daily goal: 2 hours | 📍 ~$200

- [ ] 🎯 **Take the Databricks Data Engineer Associate Exam**
- [ ] 🏆 Add certification to LinkedIn and resume
- [ ] Start **Project #1:** Build an end-to-end ETL pipeline
  - Download a free public dataset (e.g., from [Kaggle](https://kaggle.com))
  - Ingest → Clean → Transform → Store as Delta table in Databricks
- [ ] **End of week goal:** Pass the exam + start your first real portfolio project

---

## 🟣 PHASE 3: GenAI SPECIALIZATION
### *"Become the GenAI Data Engineer"* | Months 4–5

---

### Week 12 — GenAI on Databricks
> 🕐 Daily goal: 1.5–2 hours | 📍 Free

- [ ] Learn: What is RAG (Retrieval-Augmented Generation)?
- [ ] Learn: Vector search, embeddings, and LLM basics
- [ ] Explore: Databricks **Mosaic AI** tools
- [ ] Watch: Databricks YouTube tutorials on GenAI engineering
- [ ] **End of week goal:** Understand how GenAI pipelines work inside Databricks

---

### Week 13 — Build Project #2 (GenAI Project)
> 🕐 Daily goal: 2 hours | 📍 Free/Low cost

- [ ] Build: A simple **RAG chatbot** using Databricks + a free LLM API (e.g., HuggingFace)
- [ ] Steps:
  - Load documents into Databricks
  - Create vector embeddings
  - Query with a prompt and return AI-generated answers
- [ ] Document your project clearly with a README on GitHub
- [ ] **End of week goal:** Have a live GenAI demo project on your GitHub 🚀

---

### Week 14 — Polish Your Profile & Start Applying
> 🕐 Daily goal: 1.5 hours

- [ ] Set up / update your **GitHub profile** with both projects
- [ ] Update **LinkedIn:**
  - Add all Databricks certifications
  - Headline: *"Aspiring Data Engineer | Databricks Certified | GenAI Enthusiast"*
  - Add skills: SQL, Python, PySpark, Delta Lake, Databricks, GenAI
- [ ] Write your **resume** using a free template from [Canva](https://canva.com)
  - Highlight: certifications, projects, and skills — NOT degree
- [ ] **End of week goal:** Job-ready profile on LinkedIn and GitHub ✅
